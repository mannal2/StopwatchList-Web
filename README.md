# StopwatchList-Web
스톱워치를 동시에 여러 개를 운용할 수 있는 웹사이트

## 개발동기
고등학교 축제 때 이름표 때기 게임을 진행하게 되었는데 동시에 여러 명의 시간을 측정할 수 있는
프로그램이나 사이트가 필요했으나 찾지 못함. 필요성을 느끼고 직접 개발하게 됨.

## 기능소개
### 메인화면 (/)
사용자의 정보를 입력하고 시간을 측정함.

### 유저목록 (/info)
제출된 순서로 사용자들의 정보가 출력됨.
